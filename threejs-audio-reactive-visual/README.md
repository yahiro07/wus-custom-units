# ThreeJS starter

This is a general template for ThreeJS applications. It uses [Vite](https://vite.dev/) to create the bundle and [Tweakpane](https://github.com/cocopon/tweakpane) for live updates.

# Before we start
This has been developed with NodeJS `14.15.5`; it should work with other versions too, but in case something doesn't work I recommend to switch to version `14.15.5` with [nvm](https://github.com/nvm-sh/nvm).

## Setup
```shell
$ yarn install
```

## Develop

Run

```shell
$ yarn dev
```

then open a new browser window and navigate to `http://localhost:1234`
then open a new browser window and navigate to `http://localhost:5173`

## Build

```shell
$ yarn build
```
